# Task Manager Pro

## Project Description

Task Manager Pro is a web-based task management system developed using Node.js, Express.js, SQLite, HTML, CSS, and JavaScript.

## Features

* User Registration
* User Login (JWT Authentication)
* Create Tasks
* View Tasks
* Delete Tasks
* Task Statistics

## Technologies

* Node.js
* Express.js
* SQLite
* HTML/CSS/JavaScript
